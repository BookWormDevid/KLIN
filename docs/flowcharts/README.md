## How To Render

- In VS Code, open any `.md` file here and press `Ctrl+Shift+V` to open Markdown Preview.
- If Mermaid does not render in the preview, install `Markdown Preview Mermaid Support`.
- On GitHub, these Markdown files should render directly because each one uses a fenced `mermaid` block.
